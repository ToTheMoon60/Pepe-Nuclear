Pepe Nuclear Coin - Landing Page

Struktur:
- index.html  -> Hauptseite der Homepage
- images/     -> Enthält alle Bilder (Putin, Trump, Coin-Logo)

So hochladen:
1. Entpacke die ZIP-Datei.
2. Lade alle Inhalte direkt ins GitHub Repository hoch (z.B. 'Pepe-Nuclear').
3. Stelle sicher, dass unter GitHub Pages -> Settings -> Pages die Source 'main' branch und root ausgewählt ist.
4. Die Seite ist erreichbar unter: https://tothemoon60.github.io/Pepe-Nuclear/
